Simulador de Costos de Seguros

Simulador web desarrollado con HTML, CSS y JavaScript.

Permite calcular el costo estimado de seguros:
- Automotor
- Motovehículo
- Vivienda

Cada seguro tiene una cotización distinta según los datos ingresados por el usuario, como:
- Valor del bien asegurado
- Año del vehículo
- Metros cuadrados (en el caso de vivienda)

El simulador incluye:
- Validación de datos alfabéticos y numéricos
- Interacción mediante eventos
- Manipulación del DOM
- Almacenamiento de cotizaciones en LocalStorage
- Botón de contacto mediante envío de email

Proyecto realizado como segundo trabajo práctico de JavaScript.
